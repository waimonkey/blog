小马地图的“Hello, World”

开始学习小马地图API最简单的方式是看一个简单的示例。以下代码实现了创建了一个地图并设定地图显示范围(该代码可从调用示例源代码页面中复制获取)：

<!DOCTYPE html>

<html>

    <head>

       <title>HelloWorld</title>

       <meta charset="utf-8" />

       <script src="http://192.168.200.12:8585/mapapi/RMap.js"></script>

       <style type="text/css">

           body {

              padding: 0;

              margin: 0;

           }

           html,body,#map {

              height: 100%;

           }

       </style>

    </head>

    <body>

       <div id="map" class="map"></div>

       </div>

    </body>

    <script>

       //初始化地图，默认地图为在线高德地图

       var rmap = new RMap.Map('map');

       //以116.390985, 39.906358（天安门广场）为中心点坐标，地图缩放到12级

       rmap.centerZoom(116.390985, 39.906358, 12);

    </script>

</html>

 

下面我们分步向您介绍：

准备页面

首先我们设置样式，使地图充满整个浏览器窗口：

       <style type="text/css">

           body {

              padding: 0;

              margin: 0;

           }

           html,body,#map {

              height: 100%;

           }

       </style>

创建地图容器元素

地图需要一个HTML元素作为容器，这样才能展现到页面上。这里我们创建了一个div元素。

<body>

       <div id="map" class="map"></div>

       </div>

    </body>

引用小马地图API文件

       <script src="http://192.168.200.12:8585/mapapi/RMap.js"></script>

命名空间

小马地图API使用RMap作为命名空间，所有类均在该命名空间之下，比如：RMap.centerZoom、RMap. addPoint等等。

初始化地图

       var rmap = new RMap.Map('map');

初始化地图，默认地图为在线高德地图。

设置地图显示范围

       rmap.centerZoom(116.390985, 39.906358, 12);

以116.390985, 39.906358（天安门广场）为中心点坐标，地图缩放到12级。
