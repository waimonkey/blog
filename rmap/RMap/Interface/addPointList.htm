添加一组图片点

addPointList(jsonPoints,option)

根据经纬度在地图上添加点位。

参数

类型

描述

jsonPoints

String

Json格式的点数据，示例：'[{"lng":116.34,"lat":39.13,"isOnLine":0},{"lng":116.24,"lat":39.13,"isOnLine":1}]'

option

可选

 

 

id

String

该组点id，id不可重复，在移除等操作时会用到。注意：在不指定id的值时，系统会自动生成id。

 

lngField

String

经度字段，默认‘lng’

 

latField

String

纬度字段，默认‘lat’

 

pic

String

自定义默认图标的路径

 

attrPic

Array

根据属性定义图标，示例：[{"field":"isOnLine","value":1,"pic":"../a.png"},{"field":"isOnLine","value":0,"pic":"../b.png"}]

 

anchor

[Number,

Number]

图标偏移

 

isCenter

Boolean

是否缩放到该组数据，默认为false

 

data

Object

用于自定义传参

 

onClick

Function

当点击时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

onMouseMove

Function

当鼠标移入时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

onClick回调函数参数

参数

类型

描述

event

Object

 

 

lng

Number

经度

 

lat

Number

纬度

 

id

String

点位id

 

data

Object

点位属性

 

onMouseMove回调函数参数

参数

类型

描述

event

Object

 

 

lng

Number

经度

 

lat

Number

纬度

 

id

String

点位id

 

data

Object

点位属性

 

 

removePointList(pointListId,option)

移除地图上的点，如果不传参数则移除所有

参数

类型

描述

pointListId

Array.<pointList.id>|

pointList.id

地图上已添加点集合的id。如果不传则移除所有

option

可选

 

 

暂留
