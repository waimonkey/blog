轨迹播放

addTrackPlay(points,option)

播放/回放特定或关注对象在时间周期范围内的历史活动轨迹

参数

类型

描述

trackList

Array<trackObj>

轨迹点位的集合，点位是Object类型，里面必须要有lng（经度）和lat（纬度）两个字段。

option

可选

 

 

id

String

标记id，id不可重复，在移除等操作时会用到。注意：在不指定id的值时，系统会自动生成id。

 

speed

Number

控制轨迹播放速度，单位是毫秒

 

isCenter

Boolean

是否居中显示，默认为false

 

isShowArrow

boolean

是否显示箭头

 

isShowTime

boolean

是否在面板显示时间

 

showTimeKey

String

面板显示时间对应字段名

 

trackPointPic

String

轨迹点图标

 

trackPointAnchor

[Number,Number]

轨迹点图标偏移量

 

data

Object

用于自定义传参

 

mouseMoveCallback

Function

鼠标移动时执行的回调函数，回传的参数参见mouseMoveCallback回调函数参数表。示例：function(event){…}

 

playCallback

Function

轨迹回放时执行的回调函数，回传的参数参见playCallback回调函数参数表。示例：function(event){…}

 

 

 

 

trackObj轨迹参数

参数

类型

描述

trackId

String

轨迹id

pointList

Array<point>

要进行对比的点位的集合，点位是Object类型，里面必须要有lng（经度）、lat（纬度）、time（时间）三个字段。

styleOption

可选

线条样式

 

width

Number

线条的宽度

 

color

String

颜色值，示例：'#FF000'、 'red'。

 

opacity

Number

透明度，0-1。

参数

类型

可选

点的经度

 

lat

Number

点的经度

 

data

Object

点传入的属性值

参数

类型

描述

event

可选

 

 

lng

Number
