添加文字点

addTextPoint(lng,lat,text,option)

根据经纬度在地图上添加文字点位

参数

类型

描述

lng

Number

经度

lat

Number

纬度

text

String

文字内容

option

可选

 

 

id

String

文字点id，id不可重复，在点位高亮、移除等操作时会用到。注意：在不指定id的值时，系统会自动生成id。

 

font-weight

String

字体粗细（normal、bold）

 

font-size

String

字体大小（10px）

 

font-family

String

字体系列（宋体、微软雅黑）

 

data

Object

用于自定义传参

 

onClick

Function

当点击时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

onClick回调函数参数

参数

类型

描述

event

可选

 

 

lng

Number

经度

 

lat

Number

纬度

 

text

String

文字内容

 

id

String

点位id

 

data

Object

自定义的传参

removeTextPoint (pointId,option)

移除地图上的文字点，如果不传参数则移除地图上的所有文字点

参数

类型

描述

pointId

Array.<point.id>|

point.id

地图上已添加点位的id的数组集合。如果不传则移除所有

option

可选

 

 

暂留
