添加折线

addLine(points,option)

在地图上添加折线

参数

类型

描述

points

Array.<[lng,lat]>

点位坐标集合

option

可选

 

 

id

String

折线id，id不可重复，在折线移除等操作时会用到。注意：在不指定id的值时，系统会自动生成id。

 

width

Number

线条的宽度

 

color

String

颜色值，示例：'#FF000'、 'red'。

 

opacity

Number

透明度，0-1。

 

isCenter

boolean

是否缩放居中，默认为false

 

isShowArrow

boolean

是否显示箭头

 

data

Object

用于自定义传参

 

onClick

Function

当点击时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

onClick回调函数参数

参数

类型

描述

event

可选

 

 

points

Array.<[lng,lat]>

折线的节点坐标集合

 

id

String

折线id

 

clickPosition

[lng,lat]

鼠标点击处的经纬度坐标。 此信息可以在弹出气泡窗口时使用

 

data

Object

自定义的传参

removeLine(lineId,option)

移除地图上的折线，如果不传参数则移除地图上的所有折线

参数

类型

描述

lineId

Array.<line.id>|

line.id

地图上已添加折线的id的数组集合。如果不传则移除所有

option

可选

 

 

暂留
