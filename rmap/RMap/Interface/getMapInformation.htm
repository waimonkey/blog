坐标拾取

getCoordinate(callback, option)

鼠标移动时，获取坐标数据

参数

类型

描述

callback

Function

拾取完成后的回调函数，详细参数见回调参数表

示例：function(event) {…}

option

可选

 

 

showTime

Number

绘制点显示时间，单位为毫秒，设为-1一直显示

 

repeatMode

Boolean

是否重复获取，默认是true

 

回调事件参数

参数

类型

描述

event

可选

 

 

lng

Number

经度

 

lat

Number

纬度
