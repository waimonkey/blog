弹出气泡窗口

showPopWindow (lng,lat,div,option)

根据经纬度弹出容器式地图气泡窗口，气泡窗口中显示指定的div页面。这种气泡窗口在地图上只会显示一个，不能同时显示多个。

参数

类型

描述

lng

Number

经度

lat

Number

纬度

div

String

嵌入页面的地址

option

可选

 

 

isShowCloseButton

Boolean

时候显示关闭按钮，默认为true

 

isCenter

Boolean

是否居中显示，默认为false

 

isHighLight

Boolean

是否高亮显示，默认为false

 

position

Array

显示位置，单位是像素，格式为数组,默认为地图中心点，参数分别为距离屏幕左上角的像素值（isCenter为true时本参数可用）

 

autoPan

Boolean

显示气泡时是否确保当前气泡在地图范围，默认为true

 

onPopWindowClose

Function

点击气泡窗口上的关闭按钮的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

hidePopWindow(option)

隐藏地图气泡窗口

参数

类型

描述

option

可选

 

 

暂留
