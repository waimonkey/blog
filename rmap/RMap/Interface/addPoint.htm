添加图片点

addPoint(lng,lat,option)

根据经纬度在地图上添加点位。

参数

类型

描述

lng

Number

经度

lat

Number

纬度

option

可选

 

 

id

String

标记id，id不可重复，在点位高亮、移除等操作时会用到。注意：在不指定id的值时，系统会自动生成id。但在多个对象同时指定为一个id时。地图上只会加载第一个点。

 

pic

String

自定义图标的路径

 

anchor

[Number,

Number]

图标偏移

 

data

Object

用于自定义传参

 

onClick

Function

当点击标记时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

onMouseMove

Function

当鼠标移入标记时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

onClick回调函数参数

参数

类型

描述

event

Object

 

 

lng

Number

经度

 

lat

Number

纬度

 

id

String

点位id

 

data

Object

自定义的传参

 

onMouseMove回调函数参数

参数

类型

描述

event

Object

 

 

lng

Number

经度

 

lat

Number

纬度

 

id

String

点位id

 

data

Object

自定义的传参

 

 

removePoint(pointId,option)

移除地图上的标记，如果不传参数则移除地图上的所有标记

参数

类型

描述

pointId

Array.<point.id>|

point.id

地图上已添加点位的id的数组集合。如果不传则移除所有标记

option

可选
