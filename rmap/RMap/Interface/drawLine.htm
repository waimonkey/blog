绘制线

drawLine(callback,option)

绘制线

参数

类型

描述

callback

Function

绘制完成后的回调函数，详细参数见下表“回调事件参数”。示例：function(event){…}

option

可选

 

 

id

String

绘制图形的id，在删除时使用

 

width

Number

线条的宽度

 

color

String

颜色值，示例：'#FF000'、 'red'。

 

opacity

Number

透明度，0-1。

 

回调事件参数

参数

类型

描述

event

可选

 

 

id

String

绘制图形的id，在删除时使用

 

points

Array.<[lng,lat]>

线中的点

 

 

removeDrawLine(lineIds)

删除绘制的线

参数

类型

描述

lineIds

Array.<line.id>

绘制线时传入的id
