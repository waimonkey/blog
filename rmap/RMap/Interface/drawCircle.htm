绘制圆形

drawCircle(callback,option)

绘制圆形

参数

类型

描述

callback

Function

绘制完成后的回调函数，详细参数见回调参数表

示例：function(event) {…}

option

可选

 

 

id

String

绘制图形的id，在删除时使用

 

number

Number

调用一次接口绘制的图形数量，默认为1次。[X1] 

 

outLineWidth

Number

外部线条样式，宽度

 

outLineColor

String

外部线条样式，颜色值，示例：'#FF000'、 'red'。

 

outLineOpacity

Number

外部线条样式，透明度，0-1。

 

color

String

颜色值，示例：'#FF000'、 'red'。

 

opacity

Number

透明度，0-1。

 

回调事件参数

参数

类型

描述

event

可选

 

 

id

String

绘制图形的id，在删除时使用

 

extent

Object

圆形外界矩形的左下角（最小）和右上角（最大）坐标。

 

center

[lng,lat]

圆心的坐标

 

radius

Number

圆的半径，单位：米

 

polygonPoints

Array

组成多边形的点

 

 

removeDrawCircle(circleIds)

删除绘制的圆形

参数

类型

描述

circleIds

Array.<circle.id>

绘制圆形时传入的id

 

 

 [X1]不要
