添加多边形

addPolygon(points,option)

在地图上添加多边形

参数

类型

描述

pointsList

Array.<[lng,lat]>

多边形点集合

option

可选

 

 

id

String

标记id，id不可重复，在图形移除等操作时会用到。注意：在不指定id的值时，系统会自动生成id。但在多个对象同时指定为一个id时。地图上只会加载第一个。

 

outLineWidth

Number

外部线条样式，宽度

 

outLineColor

String

外部线条样式，外部颜色值，示例：'#FF000'、 'red'。

 

outLineOpacity

Number

外部线条样式，透明度，0-1。

 

color

String

颜色值，示例：'#FF000'、 'red'。

 

data

Object

用于自定义传参

 

onClick

Function

当点击标记时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}

 

onClick回调函数参数

参数

类型

描述

event

可选

 

 

pointsList

Array.<[lng,lat]>

多边形点集合

 

id

String

多边形id

 

clickPosition

[lng,lat]

鼠标点击处的经纬度坐标。 此信息可以在弹出气泡窗口时使用

 

data

Object

自定义的传参

 

removePolygon(polygonId,option)

移除地图上的多边形，如果不传参数则移除地图上的所有图形

参数

类型

描述

polygonId

Array.<polygon.id>|

polygon.id

地图上已添加图形的id的数组集合。如果不传则移除所有

option

可选

 

 

暂留
